# Excel Master Pro 📊

Analizador de presupuestos y costos en Excel.

## Despliegue en Railway

Este proyecto está configurado para desplegarse automáticamente en Railway.

### Pasos para subir a GitHub:
1. Crea un nuevo repositorio en GitHub (vacío).
2. Ejecuta los comandos que te proporcionará Antigravity para conectar tu carpeta local con GitHub.
3. Sube los cambios con `git push`.

### ¿Cómo funciona el presupuesto en la web?
He modificado la aplicación para que funcione de "forma compartida":
- Si guardas tu archivo de validación con el nombre **`presupuesto_maestro.xlsx`** en esta misma carpeta y lo subes a GitHub, la aplicación lo cargará **automáticamente** para cualquiera que abra la página.
- De esta forma, tu compañero no tendrá que subir el archivo manualmente cada vez; verá lo que tú hayas subido al servidor.

### Requisitos
- Node.js (para ejecución local si se desea).
- GitHub Account.
- Railway Account.
